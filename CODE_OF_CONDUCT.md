# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All participants in this project are expected to be respectful and constructive. Please read the full text at the link above.

## Reporting

If you experience or witness unacceptable behavior, please contact the project maintainers. All reports will be handled confidentially.
